Manage Language:
EasyDonation form has multiple language support feature. In this form you can change the static text displaying on forms as well as the dynamic data in many languages. You can add or remove any language displaying on page using xml file named "language.xml" placed under "includes/dropdownXml" directory. Please must write the value of "keyword" in language.xml file & the php files under "includes/languages" directry should be named with keyword value written in json xml file.

Now the next step is to set the default language for this form. This you can do by modifying configuration.php file. There you can see the below code: 

$defaulLanguage=array(
            "name"=>"English",
             "keyword"=>"en"  
      );

here you need to write the language name & keyword written in language.xml file.

To manage the static text of form & messages you need to look at "includes/dropdownXml" directory. There you can see file for each language where we have written constants for each text.

Add/Edit Currency:
In EasyDonation form you can add any number of currencies. To add or remove any currency you need to look at "currency.xml" file placed under "includes/dropdownXml" directory.This xml file have 3 fields "value","name" & "sign". Value keeps the 3-character ISO-4217 codes, name keeps the mane of currency that you want to display in dropdown as label, sign is currency symbol.
If you are using paypal then please keep paypal supported currencies. You can find the list of paypal supporting currencies here https://developer.paypal.com/docs/classic/api/currency_codes/

We are using minimum style-sheets for EasyDonation Form. If you want to make any modification then you can look at below files under "assets/css" directory:
